# рҹ§  AI Agent Architecture

## рҹ“Ҡ Analyst Agent

Analyzes the **last 7 days of production data** and generates operational intelligence.

Responsibilities:

- Calculates production KPIs
- Measures production growth
- Identifies scrap rate trends
- Generates executive summaries using LLM reasoning
- Signals procurement when demand increases

Example KPIs:

- Production Growth
- Scrap Rate
- Operational Downtime

---

## рҹ“Ұ Procurement Agent

Manages procurement operations based on signals generated by the analyst.

Responsibilities:

- Monitors inventory levels
- Detects low stock items
- Adjusts reorder quantities
- Groups purchase orders by vendor
- Generates professional procurement emails
- Logs decisions for governance

Example output:

- Vendor grouped purchase orders
- Cost breakdown
- Total order value

---

## рҹҡҡ Logistics Agent

Evaluates shipment reliability and identifies risks to production continuity.

Responsibilities:

- Tracks shipment schedules
- Detects delivery delays
- Assesses supply chain risk
- Generates mitigation strategies using LLM reasoning

Example insights:

- Delayed shipments
- Recommended expedited shipping
- Alternative sourcing strategies
